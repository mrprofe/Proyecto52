Sobre el creador

Hola, soy Persio Martínez.

Proyecto 52 nace con una idea simple: compartir conocimiento de datos e inteligencia artificial de forma constante, práctica y entendible. Cada episodio conecta teoría, ejemplos y aplicaciones reales para que más profesionales puedan comprender el valor de los datos.

La meta no es solo aprender herramientas, sino desarrollar criterio para tomar mejores decisiones con datos.

Catálogo

Los 13 capítulos del proyecto

Recorre la ruta completa de Proyecto 52, desde los fundamentos de los datos hasta la inteligencia artificial. Cada capítulo agrupa una etapa del aprendizaje y te lleva directamente a su playlist en YouTube.

01

Bases de Datos desde Cero (Analítica y SQL)

Fundamentos de bases de datos, conceptos esenciales, introducción a SQL y primeros pasos para analizar información.

 02

Bases de Datos Avanzadas (SQL, Transacciones y NoSQL)

Consultas avanzadas, transacciones, integridad de datos, optimización básica y fundamentos de bases de datos NoSQL.

 03

Programación para Ciencia de Datos (Python, Pandas y R)

Programación aplicada al análisis de datos usando Python, Pandas y R para limpiar, transformar y explorar información.

 04

Estadística Aplicada para Ciencia de Datos

Conceptos estadísticos clave para interpretar datos, medir patrones, analizar variabilidad y tomar mejores decisiones.

 05

Ingeniería de Datos

Procesos, pipelines, integración, transformación y preparación de datos para construir soluciones analíticas confiables.

 06

Machine Learning

Introducción al aprendizaje automático, modelos supervisados y no supervisados, entrenamiento, evaluación y predicción.

 07

Machine Learning Avanzado: Árboles, Clustering y Ensambles

Árboles de decisión, clustering, modelos de ensamble y técnicas avanzadas para mejorar el desempeño predictivo.

 08

Redes Neuronales y Deep Learning

Fundamentos de redes neuronales, entrenamiento profundo, CNN, RNN y arquitecturas modernas aplicadas a datos complejos.

 09

Procesamiento de Lenguaje Natural

Análisis de texto, embeddings, modelos de lenguaje, transformers, BERT y evolución hacia los LLMs modernos.

 10

Inteligencia Artificial

Fundamentos, agentes inteligentes, toma de decisiones y aplicaciones reales.
